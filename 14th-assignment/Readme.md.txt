What is a Database?
A database is an organized collection of data that allows easy access, management, and updating. It serves as the backbone of modern applications and systems.

Types of Databases

1. Relational Database

Data is stored in rows and columns (tables).

Example: MySQL, Posture SQL.

2. Non-Relational Database

Data is stored in formats like key-value pairs, documents, or graphs.

Example: Mongo DB, Cassandra.

DBMS (Database Management System)

A software that helps in managing databases effectively.

Examples: SQL Server, Oracle DB.
